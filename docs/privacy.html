Privacy Policy

Last updated: June 2024 · Effective immediately

TL;DR: Block Domain Free collects absolutely no data. Everything stays on your device. No servers, no accounts, no tracking — ever.

1. Who We Are

Block Domain Free is a free, open-source Chrome browser extension developed by ittermit. The extension helps users block distracting websites to improve focus and productivity.

2. Data We Collect

We collect nothing. Specifically, we do not collect, transmit, store, or share:

  • Personal information (name, email, IP address, etc.)
  • Browsing history or visited URLs
  • Your block list contents
  • Usage statistics or analytics
  • Device information
  • Crash reports or diagnostics

3. How Your Data Is Stored

All data created by the extension (your block list and focus statistics) is stored exclusively on your local device using Chrome's built-in chrome.storage.local API. This data:

  • Never leaves your browser
  • Is never synced to any external server
  • Is fully deleted when you uninstall the extension
  • Can be exported or cleared by you at any time via Settings

4. Permissions We Request

The extension requests the following Chrome permissions, each with a specific purpose:

  • storage — To save your block list and stats locally on your device
  • tabs — To read the current tab URL so we can check if it should be blocked
  • webNavigation — To intercept page loads before a blocked site renders
  • scripting — To inject the content script that performs the redirect
  • host_permissions (<all_urls>) — Required to check every URL against your block list

None of these permissions are used to collect, transmit, or process any personal data.

5. Third-Party Services

The extension itself does not connect to any third-party services or APIs. The extension's landing page uses Google Fonts for typography (loaded from fonts.googleapis.com). This is subject to Google's Privacy Policy.

6. Children's Privacy

Block Domain Free does not knowingly collect any information from anyone, including children under the age of 13. Since we collect no data at all, COPPA compliance is inherent.

7. Changes to This Policy

If we make any changes to this privacy policy, we will update the "Last updated" date at the top of this page and push the update to our GitHub repository. Continued use of the extension after changes constitutes acceptance.

8. Open Source

Block Domain Free is fully open source under the MIT License. You can inspect every line of code at any time on GitHub to verify our privacy claims:
