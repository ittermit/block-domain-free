Installation

Get started in 60 seconds

No Chrome Web Store listing yet — install directly from GitHub.

1
Clone the repository
Download the source code to your computer.
git clone https://github.com/ittermit/block-domain-free.git
2
Open Chrome Extensions
Navigate to chrome://extensions and enable Developer mode in the top right corner.
3
Load unpacked
Click Load unpacked and select the block-domain-free folder you cloned.
4
Start blocking!
Pin the extension and click its icon to start blocking distracting websites immediately.
